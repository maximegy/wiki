## Concaténation
On peux combiner des chaînes de caractère en une seule en utilisant l'opérateur `+`.
Exemple :

```python
commande("ma chaine" + " de caractère")
```

On peut aussi concaténer avec des variables:
```python
commande("bonjour " + "$user")
```
